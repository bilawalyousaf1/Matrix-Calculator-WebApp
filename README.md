# Matrix Calculator Web App

A responsive multi-page web app to perform addition, subtraction, and multiplication on 2x2, 3x3, and 4x4 matrices. Built with HTML, CSS, and vanilla JavaScript. Works fully in-browser with no external dependencies.

## Features
- Supports 2x2, 3x3, and 4x4 matrices
- Addition, Subtraction, Multiplication
- Reset button to clear inputs
- Dark-themed, responsive UI
- Works on desktop and mobile devices

## Pages
- `index.html` → 2x2 Matrix Calculator  
- `3x3.html` → 3x3 Matrix Calculator  
- `4x4.html` → 4x4 Matrix Calculator  

## Technologies
- HTML5  
- CSS3 (Flexbox & Grid, responsive)  
- JavaScript (Vanilla)

## Usage
1. Open the desired HTML page in a browser  
2. Enter numbers in Matrix A and B  
3. Click Add / Subtract / Multiply to see the result  
4. Click Reset to clear all fields
